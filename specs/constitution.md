# Constitution — 주식 공부 웹앱

## 프로젝트 목적
비전공자, 주식 공부를 막 시작한 사람이 국내 반도체 주식 시세를 보고 핵심 용어를 쉽게 학습할 수 있는 개인 학습 목적 웹앱.

## 핵심 원칙

### 1. 비전공자 친화성 최우선
- 모든 용어 설명은 금융/투자 비전공자가 이해할 수 있는 언어로 작성
- 수식보다 실생활 예시를 앞세운다
- 어려운 개념은 단계적으로 풀어서 설명

### 2. 모바일 퍼스트
- 375px 기준으로 먼저 설계하고 데스크탑으로 확장
- 하단 탭 네비게이션 (모바일) / 사이드 네비게이션 (데스크탑)
- 터치 친화적 인터랙션

### 3. 무료 데이터 소스만 사용
- Yahoo Finance 공개 API (무료, 인증 불필요)
- 실시간이 아닌 지연 시세 허용 (개인 학습 목적)

### 4. 단순함 유지
- 과도한 기능 추가 금지
- 핵심 2개 메뉴(대시보드, 용어)에 집중
- 빠른 로딩과 명확한 정보 전달 우선

## 기술 제약
- Next.js 14+ (App Router)
- TypeScript strict mode
- Tailwind CSS + shadcn/ui
- 서버 컴포넌트 활용 (Yahoo Finance API는 서버에서 호출)
