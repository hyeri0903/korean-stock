# Specification — 주식 공부 웹앱

## 사용자 스토리

### 대시보드
- 사용자는 앱을 열면 국내 주요 반도체 종목의 현재가와 등락률을 한눈에 볼 수 있다
- 사용자는 각 종목의 오늘 시가/고가/저가를 확인할 수 있다
- 사용자는 30초마다 자동으로 갱신되는 시세를 볼 수 있다
- 사용자는 최근 1개월 미니 차트를 통해 가격 흐름을 파악할 수 있다

### 주식 용어
- 사용자는 주식 관련 핵심 용어 30개+를 카드 형태로 볼 수 있다
- 사용자는 카테고리(기초/가치평가/재무/배당/차트)로 용어를 필터링할 수 있다
- 사용자는 검색창에서 용어를 검색할 수 있다
- 사용자는 용어 카드를 클릭해 상세 설명, 계산식, 실생활 예시를 볼 수 있다
- 사용자는 용어 상세 페이지에서 관련 용어로 이동할 수 있다

## 주식 용어 목록 (초기 30개)

이동평균선, 볼린저밴드, RSI, MACD, 거래량
